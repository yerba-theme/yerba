# VS Code

## Install

Search for "Yerba" in the Extensions panel (`Ctrl+Shift+X`), then select "Yerba" from the Color Theme picker (`Ctrl+K Ctrl+T`).

## Palette

| Color | Hex |
|---|---|
| Moss | `#6a9a6a` |
| Terracotta | `#a05a4a` |
| Sand | `#a89880` |
| Copper | `#b87848` |
| Flint | `#909088` |
| Background | `#181818` |
| Foreground | `#bcbcbc` |
